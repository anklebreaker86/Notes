## 2.3 字符串

字符串就是一系列字符

在Python中，用引号括起的都是字符串，其中的引号可以是单引号，也可以是双引号

```python
"This is a string."
'This is also a string'
```



**使用方法修改字符串的大小写**

```python
name = "ada lovelace"
print(name.title())
```

输出：Ada  Lovelace



还有upper和lower方法
